========================================================================
Guide to deploying and installing Plone in production
========================================================================

.. admonition:: Description

    A guide to the Plone deployment stack, including load balancing,
    proxy caching,
    server preparation, backups, log rotation, and process control.

    This guide particularly focuses on
    `Unix-like <https://en.wikipedia.org/wiki/Unix-like>`_ environments,
    though the stack discussion may be useful to everyone.
